## 🎨 Frontend Setup (ReactJS + Material-UI)

### **1️⃣ Install Dependencies**
Navigate to the root directory and install dependencies:

```
cd root_dir.
npm install
```

### **2️⃣ Start the Frontend**

```
npm start
```
